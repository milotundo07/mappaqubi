# Aggiornamento gestione dati Mappa QuBì

Carica nel repository questi quattro file:

- `index.html` sostituisce quello esistente
- `app.js` sostituisce quello esistente
- `service-worker.js` sostituisce quello esistente
- `admin.css` è un file nuovo da aggiungere nella cartella principale

Non cancellare:

- `styles.css`
- le cartelle `data`, `icons` e `strumenti`
- `manifest.webmanifest`
- `privacy.html`

## Funzioni aggiunte

1. Importazione CSV con guida dettagliata e modello scaricabile.
2. Scelta tra aggiunta ai dati esistenti e sostituzione completa.
3. Salvataggio persistente nel browser tramite localStorage.
4. Pulsante di ripristino dei dati iniziali.
5. Modulo per aggiungere un singolo punto, anche scegliendolo sulla mappa.
6. Supporto a CSV separati da virgola o punto e virgola.

## Limite importante

Le modifiche restano sul dispositivo e sul browser in cui vengono effettuate. Non vengono scritte nel repository GitHub e non si sincronizzano tra telefoni o computer.
