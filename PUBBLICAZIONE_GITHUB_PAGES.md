# Pubblicazione su GitHub Pages — guida rapida

## 1. Crea il repository

Vai su GitHub e crea un repository pubblico chiamato, per esempio:

`mappa-qubi`

## 2. Carica i file

Nel repository scegli `Add file` → `Upload files`.

Trascina tutti i file e le cartelle presenti nel pacchetto.

Controlla che `index.html` sia nella pagina principale del repository.

## 3. Attiva GitHub Pages

Apri:

`Settings` → `Pages`

Imposta:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

Premi `Save`.

## 4. Condividi il link

GitHub mostrerà un indirizzo simile a:

`https://tuonome.github.io/mappa-qubi/`

Questo è il link da inviare agli utenti.

## 5. Installazione sul telefono

Dopo aver aperto il link:

- Android/Chrome: menu del browser → `Installa app` oppure `Aggiungi a schermata Home`.
- iPhone/Safari: pulsante Condividi → `Aggiungi alla schermata Home`.

Non serve Python.
